# 🌱 Газон АкваГрин - Сайт Гидропосева

Современный адаптивный сайт для компании по гидропосеву газонов "Газон АкваГрин". Полнофункциональная веб-платформа с административной панелью и интеграцией мессенджера.

## 🚀 Возможности

### Для клиентов:
- ✅ Красивая landing page с видео и фото
- ✅ Интерактивный выбор видов газона
- ✅ Контактная форма с валидацией
- ✅ Адаптивный дизайн для всех устройств
- ✅ Быстрая загрузка и оптимизация

### Для администраторов:
- ✅ Панель управления контентом
- ✅ Просмотр заявок клиентов
- ✅ Управление видами газонов
- ✅ Статистика и аналитика

### Технические особенности:
- ✅ Интеграция с MAX Messenger
- ✅ JSON база данных
- ✅ JWT аутентификация
- ✅ Rate limiting и безопасность
- ✅ File upload поддержка

- **React 19** - Современный UI фреймворк

## 📋 Быстрый старт

### Требования:
- Node.js 16+
- npm 7+

### Установка:

```bash
# Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/gazon-aquagreen.git
cd gazon-aquagreen

# Установить зависимости
cd backend && npm install
cd ../fullstack/client && npm install
cd ../fullstack/admin && npm install

# Настроить переменные окружения
cp backend/.env.example backend/.env
# Отредактируйте .env файл

# Запустить серверы
cd backend && npm run dev              # API сервер (порт 3001)
cd ../fullstack/client && npm run dev  # Клиент (порт 5173)
cd ../fullstack/admin && npm run dev   # Админка (порт 5174)
```

## 🌐 Доступ

- **Сайт:** http://localhost:5173
- **Админка:** http://localhost:5174
- **API:** http://localhost:3001/api
